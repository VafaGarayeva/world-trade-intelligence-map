# 🌌 World Trade Intelligence Map (WTIM)

An advanced, high-performance global trade shock simulation engine and analytics dashboard. This platform models bilateral trade dependencies, predicts supply chain contagion cascades using a 3-hop network propagation math, and visualizes systemic risk profiles in a futuristic cyberpunk interface.

---

## 🚀 Core Features

- **3-Hop Contagion Graph Engine (Java Backend):** Simulates trade shock propagation through a network of 20 major economies and 74 strategic bilateral trade corridors with a `35% decay factor` per hop.
- **Dynamic Scenario Engine:** Includes 7 interactive, pre-parameterized geopolitical crisis presets (e.g., US–China Trade War, Taiwan Semiconductor Ban, Saudi Oil Embargo) alongside custom simulation inputs.
- **Automated Batch Processing:** High-speed `--batch` mode archiving that processes all preset scenarios, generates macro summaries, strips ANSI styling, and exports clean `.txt` files automatically.
- **Cyberpunk Analytics Dashboard (HTML5/CSS3/JS UI):** A futuristic dark-mode monitoring center with dynamic radar pulse positioning, real-time log streams, and modular sector risk matrix displays.
- **Statistical Visualization Suite (R Analytics):** Automated scripts generating production-ready financial risk barplots structured and optimized for institutional layout requirements.
