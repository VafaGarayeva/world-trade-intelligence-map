# World Trade Intelligence Map (WTIM) 🌍📊

An interactive, cyber-punk themed real-time global trade flow and economic risk simulation analyzer. This dashboard visualizes geopolitical stress scenarios, supply chain disruptions, and financial shocks on a dynamic SVG world map.

🔗 **Live Project Link:** [https://vafagarayeva.github.io/world-trade-intelligence-map/](https://vafagarayeva.github.io/world-trade-intelligence-map/)

---

## 🚀 Project Overview

WTIM is built for analysts, economists, and data enthusiasts to explore how localized geopolitical risks ripple across the global economy. Users can trigger macro-economic simulation events and watch impact vectors change dynamically across multiple metrics.

### ⚡ Featured Simulation Scenarios:
* **Taiwan Chip Crisis:** Simulates an 80% semiconductor disruption affecting automotive and electronics sectors globally.
* **Suez Canal Block:** Mimics a 90% choke-point blockage tracking immediate spikes in maritime freight indices.
* **Oil Supply Shock (AZE):** Analyzes infrastructure/export variances in the Caspian Basin and its power grid impact on South-Eastern Europe.
* **China Trade & Export Shock:** Evaluates a catastrophic 95% manufacturing drop and its consequences on global GDP via Monte Carlo estimations.

---

## 🛠️ Tech Stack & Key Features

* **Frontend Architecture:** Vanilla HTML5, Semantic Layout, and Modern CSS Grid/Flexbox structures.
* **Dynamic Data Engine:** Built with vanilla JavaScript, using custom orchestration logic to handle dynamic state updates without lagging.
* **SVG Data Visualization:** Fully responsive responsive custom world map layout with integrated directional flow path tracking (`<svg>` lines & text anchoring).
* **Cybernetic UI:** Neon color-coded badges, dark-mode dashboard interfaces, pulsing beacons, and structural asset styling.
